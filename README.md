### Проект: «Веб-додаток моніторингу будівельних майданчиків з AI для запобігання падінням з висоти»

**Розробник:**

**ПІБ:** Постановський Ігор

**Група:** ІПЗм(д)-25

---

## Опис проекту

Веб-додаток для моніторингу будівельних майданчиків, який використовує штучний інтелект для аналізу даних із пристроїв Інтернету речей (IoT) для запобігання аваріям, зокрема для виявлення та інформування про ризик падіння працівників з висоти.

## Структура репозиторію

- `0-BusinessGoalAnalysis/` - Аналіз бізнес-цілей (Лабораторна №1)
- `1-Requirements/` - Збір та опис вимог (Лабораторна №2)
- `2-Design/` - Проектування системи
- `2-FuncNoFuncRequirements/` - Функціональні та нефункціональні вимоги
- `3-RequirementsValidation/` - Валідація вимог
- `4-UserInterface/` - Користувацький інтерфейс

## Технологічний стек

- **Backend:** Azure App Service, Azure Functions
- **IoT:** Azure IoT Hub
- **Storage:** Azure Blob Storage
- **AI/ML:** Azure Machine Learning
- **Frontend:** Інтерактивні панелі керування (UX/UI)
