## Примітки

- **NFR** (Non-Functional Requirements) - нефункціональні вимоги прив'язані до відповідних **FR** (Functional Requirements)
- Кількісні показники визначені відповідно до стандарту ISO/IEC/IEEE 29148:2011
- Вимоги охоплюють категорії: продуктивність, надійність, безпека, юзабіліті, масштабованість, сумісність, доступність, переносимість, підтримка, стійкість до збоїв, енергоефективність, юридичні та організаційні аспекти
