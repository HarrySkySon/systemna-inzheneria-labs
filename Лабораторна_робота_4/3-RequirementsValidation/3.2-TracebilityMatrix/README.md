## Легенда

| Позначення | Опис |
|------------|------|
| **FR№** | Functional Requirement - Функціональна вимога |
| **NFR№** | Non-Functional Requirement - Нефункціональна вимога |
| **Epic №** | Епік - велика група функціоналу |
| **US№** | User Story - Користувацька історія |
| **AC№** | Acceptance Criteria - Критерії приймального тестування (Gherkin) |
