# Діаграма бізнес-цілей, проблем та критеріїв успіху

## Опис діаграми

Діаграма показує взаємозв'язок між:
- **Бізнес проблемою** - висока аварійність через падіння з висоти
- **Бізнес цілями** - 6 цілей (3 фінансові, 3 нефінансові)
- **Рішеннями** - технічні рішення для досягнення цілей
- **Критеріями успіху** - метрики для оцінки досягнення цілей

---

## Бізнес проблема

Висока аварійність на будівельних майданчиках через падіння працівників з висоти становить понад 40% всіх нещасних випадків у будівельній галузі. Це призводить до:
- Високих витрат на компенсації
- Втрат робочого часу
- Репутаційних ризиків
- Штрафів від регуляторів

---

## Бізнес цілі

### Фінансові цілі:

1. **Зменшення компенсаційних витрат**
   - З $500,000 до $150,000/рік
   - Економія: $350,000/рік

2. **ROI 200% за 18 місяців**
   - Очікувані інвестиції: $300,000
   - Очікуваний прибуток: $600,000

3. **Зниження страхових витрат**
   - Заощадження: $200,000/рік
   - Завдяки покращенню показників безпеки

### Нефінансові цілі:

4. **Задоволеність працівників 85%**
   - Термін: 6 місяців
   - Метод вимірювання: quarterly survey

5. **Сертифікація ISO 45001**
   - Термін: 12 місяців
   - Міжнародний стандарт безпеки праці

6. **Покращення репутації**
   - Стати галузевим лідером з безпеки
   - Залучення кращих фахівців

---

## Технічні рішення

### Рішення 1: IoT датчики моніторингу
- GPS/RFID трекери для працівників
- Датчики висоти
- Датчики руху та положення тіла

### Рішення 2: AI/ML модель прогнозування
- Аналіз історичних даних
- Виявлення патернів небезпечної поведінки
- Прогнозування ризиків у реальному часі

### Рішення 3: Система сповіщень
- Push-нотифікації на мобільні пристрої
- Звукові та візуальні попередження
- Автоматичне оповіщення служби безпеки

---

## Критерії успіху

| Критерій | Цільове значення | Метод вимірювання |
|----------|-----------------|-------------------|
| Точність виявлення ризиків | ≥ 95% | Тестування на історичних даних |
| Час реакції | < 2 секунди | Моніторинг системи |
| Покриття моніторингу | ≥ 98% робочих зон | GPS mapping |
| Uptime системи | ≥ 99.5% | Azure Monitor |
